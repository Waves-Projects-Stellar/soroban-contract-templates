### soroban-contract-templates

A curated collection of secure, tested, and reusable smart contract templates designed to accelerate Soroban development.

---

### ✨ Features

* Token templates (ERC20-like)
* Escrow contracts
* DAO governance modules
* NFT primitives
* Deployment-ready configurations

---

### 🧱 Module Breakdown

* **Core Templates**

  * Token
  * Escrow
  * Multisig

* **Advanced Templates**

  * DAO governance
  * Auction systems
  * NFT marketplace

* **Dev Tools**

  * CLI scaffolding
  * Deployment scripts

---

### 🏗 Technical Architecture

* Contracts: Rust + Soroban SDK
* Frontend: Docs site (Next.js)
* Backend: Node.js CLI tools

---

### 🌊 Drips Wave Program

Designed as a **template factory for contributors**.
